Chimp Bear - C++

Is it a Chimp or a Bear? Pacman like 2D game using C++

My first C++ project. This program was the foundation in which I began learning C++.
This pacman like game features basic techniques such as object colliders & hitboxes, AI & pathing, sprite creation & animation, audio implementation and menu navigation.
To go from nothing to being able to produce a responsive and functioning program was incredibly rewarding, and as my abilities progressed I was able to troubleshoot issues and fix bugs and incorrect logic quickly and competently. I will continue to add features to this program as my knowledge progresses , and I look forward to seeing where this project takes me.
